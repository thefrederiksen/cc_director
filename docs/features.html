Why Use CC Director?

Multiple Projects, One Window

Run multiple Claude Code sessions simultaneously across different repositories. Switch between projects instantly without losing context or terminal history.

Real Console, Not Emulation

Uses native Windows ConPTY for true console compatibility. Full support for colors, cursor control, and interactive programs - no terminal emulation quirks.

Sessions Survive Restarts

Close CC Director and your Claude sessions keep running. Reopen later and pick up exactly where you left off with full session recovery.

Real-Time Activity Monitoring

See exactly what Claude is doing at a glance. Color-coded status badges show when Claude is working, waiting for input, or needs permission.

Git-Aware Workflows

Built-in source control panel shows staged/unstaged changes, branch status, and sync state. Click files to open in VS Code instantly.

Enterprise-Grade Logging

Comprehensive diagnostic logging for debugging and auditing. Every operation is tracked with timestamps and context for full observability.
